# ✅ רשימת בדיקות מהירה - Quick Testing Checklist

## 🚀 התחל כאן!

### לפני הבדיקות:
- [ ] האפליקציה רצה על אנדרואיד
- [ ] Firebase Console פתוח (לעקוב אחרי לוגים)
- [ ] יש לך 2 מכשירים/משתמשים (לבדיקת התראות)

---

## 📋 רשימת בדיקות (5 דקות)

### 1. אימות משתמש ⏱️ 2 דקות
- [ ] הרשמה עם אימייל וסיסמה
- [ ] התחברות עם Google
- [ ] יצירת משתמש אנונימי

### 2. יצירת Hub ⏱️ 1 דקה
- [ ] צור Hub חדש
- [ ] ודא שהוא מופיע ברשימה

### 3. יצירת Game + פוסט אוטומטי ⏱️ 2 דקות
- [ ] צור משחק חדש בהאב
- [ ] **חשוב!** לך לפיד של ההאב
- [ ] **בדוק**: האם מופיע פוסט "משחק חדש נוצר ב-[שם ההאב]!"?

### 4. התראות פוש ⏱️ 3 דקות
- [ ] שלח הודעה בהאב צ'אט (משתמש 1)
- [ ] **חשוב!** במכשיר אחר/משתמש אחר (משתמש 2)
- [ ] **בדוק**: האם משתמש 2 קיבל התראת פוש?

### 5. חיפוש מגרשים ⏱️ 1 דקה
- [ ] פתח מסך חיפוש מגרשים
- [ ] חפש "מגרש כדורגל תל אביב"
- [ ] **בדוק**: האם החיפוש מחזיר תוצאות?

---

## ❌ אם משהו לא עובד:

### פוסט לא מופיע בפיד?
1. בדוק Firebase Console → Functions → Logs → `onGameCreated`
2. בדוק Firestore → feed → האם יש פוסט עם `type: "game_created"`?

### התראה לא נשלחת?
1. בדוק Firebase Console → Functions → Logs → `onHubMessageCreated`
2. בדוק Firestore → hubs → [hubId] → members → האם יש `fcmToken`?

### חיפוש לא עובד?
1. בדוק Firebase Console → Functions → Logs → `searchVenues`
2. בדוק Environment Variables → האם `GOOGLE_PLACES_API_KEY` מוגדר?
