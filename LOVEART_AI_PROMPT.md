# פרומפט ל-loveart.ai - יצירת אייקונים ותוכן גרפי ל-Kickadoor

## הקשר כללי
אני מפתח אפליקציה בשם **Kickadoor** - רשת חברתית לשחקני כדורגל שכונתיים בישראל. האפליקציה מאפשרת למצוא שדות, לארגן משחקים, לנהל קבוצות (Hubs), ולעקוב אחר ביצועים.

## סגנון עיצוב
- **סגנון**: Line icons עקבי, 2.5-3pt stroke
- **צבעים**: פלטת Kickadoor (ירוק כהה #2D5016, ירוק בהיר #1DE9B6, כתום #FF6B35, כחול #0D47A1, סגול #8E24AA)
- **רקע**: שקוף (PNG עם אלפא)
- **גודל**: 1024x1024px לכל אייקון
- **פורמט**: PNG עם רקע שקוף

## הערות נוספות

1. **עקביות**: כל האייקונים צריכים להיות באותו סגנון, עובי קו זהה, ופרופורציות דומות
2. **זיהוי**: האייקונים צריכים להיות ברורים וקלים לזיהוי גם בגודל קטן (24-32px)
3. **רקע**: חייב להיות שקוף לחלוטין (alpha = 0)
4. **מרכז**: האייקון צריך להיות ממורכז בתוך הקנבס 1024x1024
5. **מרווחים**: להשאיר מרווח מספיק מהקצוות (padding ~10-15%)
