## ğŸ¨ Product Principles

### 1. Community First
- Hubs are the heart of Kattrick
- Every feature should strengthen community bonds
- Social features are as important as game features

### 2. Mobile First
- Designed for on-the-go usage
- Quick actions (join game = 2 taps)
- Offline support (view schedules without connection)
- Push notifications for important events

### 3. Israeli Market Focus
- Hebrew interface (primary)
- English support (secondary)
- Integration with Israeli venues (Google Maps)
- Local football culture (5v5, 7v7 most common)

### 4. Simplicity Over Features
- Don't overwhelm users
- Progressive disclosure (advanced features hidden)
- Onboarding should take < 2 minutes
- Core flow: Join Hub â†’ See Games â†’ Tap to Join

### 5. Data Privacy
- Users control their profile visibility
- Can block other users
- Minors (13-18) have extra privacy by default
- Hubs are semi-private (not public directory)

## ğŸ” Privacy & Safety

### User Safety

**Blocking System:**
- Users can block others
- Blocked users can't see profile/activity
- Hub managers can ban problematic members

**Reporting:**
- Report inappropriate posts/comments
- Report abusive behavior
- Hub managers review reports

**Minors Protection:**
- Age verification required
- Minors (13-18) have restricted profiles by default
- Parents can request account deletion

### Data Privacy

**What We Collect:**
- Email, name, date of birth
- Location (for Hub discovery only)
- Game participation & stats
- Usage analytics

**What We DON'T Collect:**
- Precise GPS tracking
- Messages (stored encrypted)
- Payment info (no payments yet)

**User Rights:**
- View all their data
- Export data
- Delete account (anonymization)
